# US Illustrations Platform

Full-stack children's book illustration management platform. AI-powered character generation, illustration production, customer review, and delivery.

**Live:** https://studio.usillustrations.com

## Quick Start

```bash
npm install
npm run dev        # http://localhost:3000
```

Admin: `/admin` → login with `ADMIN_USERNAME` / `ADMIN_PASSWORD` from `.env.local`
